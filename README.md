# Proyecto Final - Tienda Web

Breve: Sitio de ejemplo con lista de productos, contacto mediante Formspree, carrito persistente y diseño responsivo.

Instrucciones rápidas:

1. Reemplazar el `action` del formulario en `index.html` por tu endpoint de Formspree: `https://formspree.io/f/your-form-id`.
2. Subir los archivos a Netlify o GitHub Pages. En GitHub Pages sube al branch `main` y activa Pages en la configuración.
3. Abrir `index.html` en el navegador para pruebas locales.

Tecnologías: HTML5 semántico, CSS (Flexbox y Grid), JavaScript (fetch, DOM, localStorage).
